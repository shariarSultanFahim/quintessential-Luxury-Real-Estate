# Quintessential - Luxury Real Estate
## This project is for Quintessential, a real estate website showcasing luxury properties like penthouses, beachfront estates, private islands, villas, and mansions.


# Key Features:

* **Seamless User Interface:** The website boasts a user-friendly and intuitive interface, making it easy for users to navigate and find their dream property.

* **Firebase Integration:** Firebase provides a robust backend for managing property listings, user accounts, and real-time data updates.

* **Property Listings:** The website displays a comprehensive list of properties with essential details and captivating visuals.

* **Property Details Page:** Each property has a dedicated page showcasing detailed information, high-quality photos.

* **Eye Catching Design:** Tailwind CSS and DaisyUI are leveraged to create a visually stunning and modern website that reflects the essence of luxury living.



# Live Site:

## Visit the live website at https://quintessentialhousing.surge.sh/



# Technologies Used:

* **Node.js and npm:** The project utilizes Node.js as the runtime environment and npm for package management.

* **React:** The website is built using React, a popular JavaScript library for building user interfaces.

* **React Router Dom:** This package enables navigation within the single-page React application.

* **Firebase:** Firebase by Google provides backend services like authentication, database (Firestore), and storage.

* **Tailwind CSS:** A utility-first CSS framework allowing for rapid and responsive design.

* **DaisyUI:** A pre-built component library built on top of Tailwind CSS, offering pre-styled components for faster development.

* **React Icons:** Provides a vast library of SVG icons for enhancing the user interface.
